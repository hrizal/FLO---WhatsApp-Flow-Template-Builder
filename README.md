# FLO - Modern WhatsApp Flow Designer

![FLO Preview](hero-preview.png)

**FLO** adalah desainer visual tercanggih untuk membuat WhatsApp Flows tanpa perlu menulis kode JSON secara manual. Bangun pengalaman percakapan yang interaktif, premium, dan profesional hanya dalam hitungan menit.

## âœ¨ Fitur Unggulan

- **Visual Designer**: Susun komponen seperti `TextInput`, `DatePicker`, `CheckboxGroup`, hingga `ImageCarousel` dengan sekali klik.
- **Real-Time Preview**: Lihat tampilan dan rasakan pengalaman alur (flow) Anda langsung di simulator perangkat mobile.
- **Multi-Screen Support**: Buat alur yang kompleks dengan banyak layar dan pengaturan navigasi yang cerdas.
- **Smart Data Binding**: Hubungkan desain Anda dengan data dinamis menggunakan sistem token `${data.variable}`.
- **Auto-Terminal Logic**: Sistem cerdas yang otomatis mendeteksi status "terminal" layar untuk memastikan Flow Anda selalu valid.
- **Markdown Editor**: Personalisasi teks Anda dengan dukungan Markdown yang kaya untuk hasil yang lebih estetik.
